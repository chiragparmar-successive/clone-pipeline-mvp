1) Section order and presence
2) Critical content missing
3) CTA text/link mismatch
4) Repeated sections or missing footer/header
5) Obvious layout regressions

OUTPUT FORMAT:
{
  "status": "pass|needs-fix",
  "issues": [
    {
      "severity": "high|medium|low",
      "type": "layout|content|data-binding",
      "description": "..."
    }
  ]
}

SCORING RUBRIC:

- `needs-fix`: any high severity issue OR 3+ medium issues.

1) Data-binding failures
2) Missing critical sections
3) Broken CTA/link behaviors
4) Visual polish mismatches
